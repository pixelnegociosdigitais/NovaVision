# PRD - Nova Vision

Crie uma plataforma SaaS premium chamada “Nova Vision”.

A plataforma deve possuir visual moderno, tecnológico, elegante e altamente profissional.

O sistema será uma plataforma de inteligência empresarial focada em monitoramento de novas empresas brasileiras utilizando dados públicos da Receita Federal/Brasil.IO.

A estética visual deve seguir referências modernas similares a:
- Linear
- Stripe
- Vercel
- Notion
- Supabase

O design deve ser:
- clean
- sofisticado
- premium
- dark mode elegante
- altamente responsivo
- preparado para grande volume de dados
- arquitetura modular e escalável

## Stack da aplicação:
- Next.js
- React
- Tailwind
- Supabase
- TypeScript
